## Запуск

### Способ 1: Docker Compose (рекомендуется)

```bash
cd deploy
docker-compose up --build -d
```

Приложение будет доступно по адресу http://localhost:8000

Для завершения:
```bash
 docker-compose down -v
```

### Способ 2: Ручной запуск

Требования:
- Python 3.11+
- PostgreSQL

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` в корне проекта (см. `.env.example`)

3. Запустите приложение:
```bash
python main.py
```

Скрипт автоматически:
- Проверит подключение к БД
- Выполнит миграции
- Запустит сервер на http://127.0.0.1:8000

## Проверка функционала

Для входа используйте учётную запись администратора:
- Email: `admin@example.com`
- Пароль: `password123`

Под администратором создайте несколько пользователей с разными ролями (ученик, преподаватель). 
При создании система выдаст временный пароль, сохраните его.

Для проверки разграничения доступа перелогиньтесь под созданными пользователями. 
Каждая роль имеет свой набор доступных функций.
